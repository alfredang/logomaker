# Logo Maker

A modern, minimalist web tool: **describe a logo in plain words and get a clean
vector logo** you can download as **SVG** or **PNG**.

Logos are generated by Claude through the **[Claude Agent SDK]** using your **Claude
subscription** — no API key required.

## How it works

- **Frontend** (`public/`) is pure HTML / CSS / JavaScript — no frameworks, no build step.
- **Backend** (`server.js`) is a tiny Node.js server. It serves the frontend and exposes
  one endpoint, `POST /api/generate`, which calls the Claude Agent SDK and asks Claude to
  return a single self-contained SVG.
- A Node server is required because the Agent SDK runs in Node and authenticates with your
  Claude subscription (the browser can't do either safely).
- The SVG renders in the browser; **PNG** is produced client-side by drawing the SVG onto a
  `<canvas>` at 1024×1024.

## Prerequisites

- **Node.js 18+** (developed on Node 22).
- The **`claude` CLI** installed and logged in to your Claude subscription
  (`claude` — then sign in). The Agent SDK reuses that login automatically.

## Run

```bash
npm install
npm start
```

Then open **http://localhost:3000**.

### Configuration

| Env var      | Default | Description                                              |
| ------------ | ------- | -------------------------------------------------------- |
| `PORT`       | `3000`  | Port to serve on.                                        |
| `LOGO_MODEL` | `sonnet`| Claude model alias (`sonnet`, `opus`, `haiku`, `fable`). |

Example: `LOGO_MODEL=opus PORT=8080 npm start`

## Usage

1. Type a description, e.g. *"a minimalist mountain peak inside a circle, single line,
   monochrome"*.
2. Click **Generate** (or press ⌘/Ctrl + Enter).
3. **Download SVG** (vector) or **Download PNG** (raster), or **Regenerate** for a new take.

[Claude Agent SDK]: https://www.npmjs.com/package/@anthropic-ai/claude-agent-sdk
